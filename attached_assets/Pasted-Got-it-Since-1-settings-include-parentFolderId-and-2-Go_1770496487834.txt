## STEP 3: Create a Google Drive folder when a Project is created

### 3.1 Database changes

**Projects table**

### 3.2 Add a Google Drive service helper

Create: `server/services/googleDrive.ts`

It must:

* load the stored refresh token/access token (whatever you already saved in step 2)
* create an authenticated Drive client using `googleapis`
* create a folder under a parent folder

**Function signatures to implement:**

* `getDriveClient(): drive_v3.Drive`
* `createProjectFolder({ parentFolderId, projectName, projectId }): Promise<{ folderId, webViewLink }>`
* `ensureProjectFolder(projectId): Promise<Project>` (creates if missing)

Folder name format:

* `${projectName} - ${projectId}` (ensures uniqueness)

Drive API calls:

* `drive.files.create` with:

  * `mimeType: "application/vnd.google-apps.folder"`
  * `parents: [parentFolderId]`
  * `fields: "id, webViewLink"`

If your Drive call needs it, include:

* `supportsAllDrives: true`
* `includeItemsFromAllDrives: true`

---

### 3.3 Hook folder creation into Project creation

Wherever you create a project (example: `POST /api/projects`):

1. Create the project in DB first (so you have projectId)
2. If Google Drive integration enabled and `parentFolderId` exists:

   * set `driveSyncStatus = "PENDING"`
   * call `createProjectFolder(...)`
   * save `googleDriveFolderId`, `googleDriveFolderUrl`, set `driveSyncStatus = "OK"`
3. If it fails:

   * set `driveSyncStatus = "ERROR"`
   * save `driveSyncError = error.message`
   * DO NOT fail the entire project creation (don’t block the user)

Create:

* `POST /api/projects/:id/drive/retry`

Logic:

### 4.2 Extend the Google Drive service: upload file into a folder

Add in `googleDrive.ts`:

`uploadFileToProjectFolder({ projectId, localFilePath, originalName, mimeType }): Promise<{ fileId, webViewLink }>`

Steps:

1. Load project from DB
2. If `project.googleDriveFolderId` is missing:

   * call `ensureProjectFolder(projectId)` first
3. Upload:

   * `drive.files.create({
        requestBody: {
          name: originalName,
          parents: [folderId]
        },
        media: {
          mimeType,
          body: fs.createReadStream(localFilePath)
        },
        fields: "id, webViewLink"
     })`

Also include `supportsAllDrives: true` if needed.
